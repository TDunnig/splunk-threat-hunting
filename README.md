# Splunk Threat Hunting Queries

Advanced Splunk queries for proactive threat hunting and incident detection.

## Query Categories

### Credential Compromise
- Monitor successful logins from unusual locations
- Detect brute force attacks across multiple accounts
- Identify impossible travel scenarios
- Track Kerberos anomalies

### Network Reconnaissance
- Port scanning detection
- DNS exfiltration attempts
- Protocol anomalies
- Traffic to known malicious IPs

## Usage
All queries are ready to copy into Splunk Search or create as scheduled alerts.

## Performance
- Index time: <2s for most queries
- Optimized with datamodel acceleration
- Compatible with Enterprise Security content
